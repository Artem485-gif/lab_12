# lab_12
ТЕМА: Використання Python бібліотек для розв'язання різних завдань
Мета: аналіз можливостей застосування бібліотек Python для обробки даних у межах заданої тематики.
Варіант 1: Збір і обробка даних з HTML-сторінок із застосуванням модуля requests та бібліотеки BeautifulSoup. Отримання даних із вебсайтів, аналіз HTML-структури та формування локальної бази даних.
